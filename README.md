# SWAT — Swift Work and Travel

Каталог сезонных офферов в США и калькулятор чистого заработка для студентов
из Центральной Азии.

## Зачем это

Студент видит ставку, умножает на часы и недели, получает красивую сумму
и покупает билет. Потом из неё уходят жильё, еда, налоги, перелёт и стоимость
программы — и лето заканчивается в ноль.

По модельному расчёту **10 направлений из 12 убыточны, если работа одна.**
Вторая одобренная работа добавляет $760–2 470 за тот же сезон — это больше,
чем разница между всеми штатами вместе взятыми.

Отсюда и продукт, и клин: SWAT показывает расчётку до оплаты, оформляет
вторую работу заранее через спонсора (иначе она незаконна и ведёт к отчислению
с программы) и говорит «сюда ехать не надо», когда цифры этого требуют.
Отговорить от плохого оффера — это фича, а не баг.

Полный разбор рынка, регуляторики, конкурентов и календаря сезона —
в [стратегическом брифинге](docs/2026-07-20-стратегический-брифинг.md).

## Настройка приёма заявок

Скопируй `.env.example` в `.env.local` и заполни. Пока переменные не заданы,
форма отвечает ошибкой 503 — намеренно, чтобы потеря заявок не осталась
незамеченной.

```
TELEGRAM_BOT_TOKEN=...
TELEGRAM_CHAT_ID=...
```

Как получить значения — в комментариях внутри [.env.example](.env.example).

## Деплой

Vercel, бесплатный тариф:

```bash
npx vercel
```

Переменные окружения задаются в настройках проекта на Vercel
(Settings → Environment Variables), а не в репозитории.

## Состояние данных

**Числа в [src/lib/states.ts](src/lib/states.ts) и
[src/lib/scenarios.ts](src/lib/scenarios.ts) помечены `verified: false`.**
Это предварительные оценки: ставки, стоимость жилья и длительность сезонов
ещё не сверены с вакансиями работодателей и данными по аренде.

Продукт продаёт точность расчёта, поэтому непроверенное число не имеет права
выглядеть как проверенное. Интерфейс показывает такие данные с оговоркой,
и её нельзя убирать до фактической сверки.

Что нужно подтвердить в первую очередь:

- минимальные ставки штатов на 2026 год;
- федеральные налоговые ступени для нерезидента на 2026 год;
- реальную стоимость жилья за неделю по каждому курорту;
- фактические ставки в вакансиях J-1 у крупных работодателей;
- стоимость перелёта из Алматы по направлениям.

## Что это не

SWAT не спонсор программы, не работодатель и не иммиграционный консультант.
Размещение участников идёт через назначенного спонсора BridgeUSA. Расчёты
на сайте — оценка, а не оффер и не налоговая консультация.
